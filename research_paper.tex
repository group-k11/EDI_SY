\begin{abstract}
Network Intrusion Detection Systems (NIDS) function at the frontier of cybersecurity, challenged by the necessity to recognize established attack vectors while robustly flagging novel, anomalous patterns that deviate from baseline topologies. This paper presents a complete, production-ready NIDS prototype integrating multiple complementary detection philosophies. Our system evaluates real-time traffic leveraging the CICIDS2017 feature schemas, deploying an ensemble of algorithmic pathways: a robust supervised Random Forest classifier for high-precision detection of historically apparent attacks, unsupervised Deep Autoencoders and Isolation Forests constrained purely on benign network operations to uncover zero-day structural anomalies, and a Long Short-Term Memory (LSTM) network to characterize temporal bursts and time-series dependencies over moving windows. The terminal identification is resolved via a hybridized, weighted ensemble engine scoring architecture bridging classical machine learning with advanced deep learning outputs. Critically, we encapsulate the intelligence within a resilient operational framework extending beyond theoretical accuracy. The platform incorporates a native Population Stability Index (PSI) Concept Drift Monitor adjusting inference thresholds autonomously against mutating operational realities, effectively mitigating the false positive bloat frequently encountered in deployed ML pipelines. Further refinement is executed by an asynchronous Threat Intelligence ingestion loop injecting real-time local IP reputation, geolocation, and ASN validations. This paper details the reproducible data preprocessing, explicit feature-locking contracts, API deployment parameters via FastAPI, and presents a dynamic React/Vite-based SOC dashboard providing analysts with instantaneous interpretation capabilities. Experimental baselines confirm that while Random Forest delivers unmatched precision on standardized attacks, the unguided ensemble branches critically protect overarching network resilience from non-stationary deviations and uncatalogued threat deployments.
\end{abstract}

\section{Introduction}
Modern networking infrastructure forms the backbone of global operations, facilitating communications across distributed IoT devices, cloud workloads, remote-worker connections, and critical microservices. As the volume and diversity of legitimate structural traffic exponentially increase, malicious entities continually innovate, employing sophisticated polymorphic payloads, stealthy distributed denial of service (DDoS) frameworks, automated infiltration scripts, and adversarial evasion techniques to bypass standardized perimeter protections. Conventionally, Intrusion Detection Systems (IDS)—such as Snort and Suricata—have heavily governed these boundaries relying inherently on manually prescribed heuristic signatures. While incredibly efficient at mitigating previously mapped threats, signature architectures suffer from foundational rigidity; they intrinsically overlook zero-day deviations lacking predefined catalog sequences \cite{khraisat2019survey, buczak2016}.

Implementing Artificial Intelligence (AI) and Machine Learning (ML) directly addresses signature rigidity by learning statistically discriminative boundaries directly from extracted flow features. Supervised classical algorithms including Random Forests (RF), Support Vector Machines (SVM), and Extreme Gradient Boosting (XGBoost) successfully map network metadata into robust decision surfaces. Despite substantial benchmark accuracy, supervised architectures introduce vulnerabilities reliant entirely upon label distributions: if a novel threat significantly shifts from the training distribution, traditional classifiers assign catastrophic confidence values to malicious events \cite{ahmad2021survey}.

Unsupervised systems and sequential deep learning architectures mitigate these precise drawbacks by evaluating alternate dimensions of network payloads. Deep Autoencoders function fundamentally by learning highly compacted dimensionality representations of exclusively benign traffic, flagging deviations via excessive reconstruction errors \cite{mirsky2018kitsune}. Additionally, Recurrent Neural Networks, specifically Long Short-Term Memory (LSTM) structures, chronologically track sequential burst phases, capturing underlying timing mechanics completely invisible to single-flow packet evaluations \cite{hochreiter1997lstm}. Importantly, these unsupervised mechanisms do not intend to replace static supervision but instead formulate highly collaborative parallel validation nodes.

This paper proposes an expansive, adaptive Hybrid AI NIDS prototype resolving classical isolation flaws. By bridging an ensemble of distinct ML/DL models with dynamic real-world environment controllers (Concept Drift limits and dynamic Threat Intelligence blocklists), the research establishes a deployable architecture translating academic inference into an interactive operational Security Operations Center (SOC) framework.

The primary contributions of this paper include:
\begin{enumerate}
    \item \textbf{Decision-Level Hybrid Architecture:} Constructing a compartmentalized parallel network unifying supervised Random Forest probabilities, Autoencoder reconstruction gradients, Isolation Forest outlier mechanics, and LSTM sequence patterns independently.
    \item \textbf{Adaptive Concept Drift (PSI):} Implementation of a real-time Population Stability Index (PSI) monitor capable of analyzing tumbling distribution shifts and dynamically restricting ML inference thresholds to counter active network concept drift.
    \item \textbf{Threat Intelligence Enrichment:} Integration of real-time IP analytics (e.g., ASN, regional geolocation, allowlist/blocklist mapping) to functionally scale algorithm severity matrices manually.
    \item \textbf{Operational Prototype Integration:} Packaging the holistic statistical engine into a scalable analytical backbone utilizing FastAPI for sub-millisecond inference and a granular React framework for dynamic dashboard interactions over streaming WebSockets.
    \item \textbf{Deployment Limitations Framework:} Acknowledging explicitly the operational discrepancies linking raw socket integrations in continuous daemon tracking versus batch CICIDS2017 flow-level feature extraction dependencies.
\end{enumerate}

Finally, Section V delivers the Conclusion.

\section{Literature Review}
\subsection{Intrusion Detection Datasets (CICIDS2017)}
Traditional validation was historically performed over the DARPA KDD Cup 99 and subsequently NSL-KDD benchmarks. While foundational, modern traffic flows render them fundamentally obsolete. The CICIDS2017 dataset, structured effectively utilizing the CICFlowMeter pipeline, provides modernized heterogeneous attacks embedded transparently against realistic global communication behaviors \cite{sharafaldin2018toward}. Its complex representation of DoS schemas, Web Attacks, and stealthy infiltration campaigns positions it as the standardized benchmark mapping contemporary NIDS capabilities. 

\subsection{Supervised Machine Learning}
Tree-based aggregations regularly dictate supervised tabular research. The Random Forest architecture iteratively reduces predictive variance by deploying parallelized decision trees against randomized bootstrap aggregations \cite{breiman2001random}. XGBoost concurrently maximizes gradient mappings, incorporating heavy penalty scaling to control branch complexity. Across CICIDS2017 schemas, ensemble tree classifiers heavily outweigh Neural Networks mapping pure parametric regressions, establishing the baseline requirement utilized within this integration \cite{chen2016xgboost}.

\subsection{Deep Learning and Unsupervised Reconstructions}
To solve zero-day vulnerabilities lacking supervised labels, the Autoencoder paradigm targets strict mathematical representations of normative sequences. The Kitsune network specifically deployed ensembles of localized autoencoders targeting raw network packets directly, extracting reconstruction deltas sequentially \cite{mirsky2018kitsune}. This research validates the operational capability to bypass labeling restrictions by restricting deep-layer optimizations purely against healthy communication protocols, assuming malicious actions induce non-conforming structural paths natively. Furthermore, recent comparative studies \cite{ferrag2020, gamage2020} emphasize that while deep learning consistently excels in mapping high-dimensional non-linear attack boundaries, standard benchmark evaluations strictly require hybridized environments to fully address adversarial evasion and modern encryption realities.

\subsection{Temporal Sequence Models}
Traffic attacks rarely occur singularly; instead, they operate as sustained structural bursts. LSTMs theoretically solve the vanishing gradient paradox affecting generalized Recurrent Neural Networks, authorizing sustained short-term memory dependencies referencing moving statistical sequences \cite{hochreiter1997lstm}. Adapting LSTM architectures to networking requires capturing adjacent temporal buffers, allowing the recognition of progressive scanning algorithms frequently bypassed by static one-shot flow analyzers \cite{nids_lstm}.

\subsection{Concept Drift Tracking}
Standard deployment paradigms incorrectly assume the network distribution remains theoretically stationary. Over months, application traffic evolves natively, heavily degrading classical detection reliability (Concept Drift). Advanced integrations deploy monitoring mechanics, assessing Kullback-Leibler (KL) divergences or Population Stability Index (PSI) coefficients to track topological shifts bridging static baseline training distributions against streaming production volumes, ensuring continuous statistical alignment \cite{psi_concept_drift}.

\subsubsection{Mathematical Cleaning and Winsorization}
Direct translations of the CICFlowMeter operations result natively in multiple infinite properties specifically triggered around time-based properties (bytes-per-second ratios encountering near-zero flow durations). \subsubsection{Dynamic Feature Selection}
High dimensionality dilutes model generalization. We apply intensive 3-tier selective methodologies:
\begin{enumerate}
    \item \textbf{Variance Limiting:} Stripping categorical elements demonstrating minimum standard scale movement bounds under 0.01 thresholds.
    \item \textbf{Correlation Clipping:} Purging natively redundant flow properties demonstrating redundant Pearson coefficient overlap $\rho \geq 0.95$. 
    \item \textbf{Boosting Extractors:} Operating primary Tree-based gradient structures extracting the Top-40 defining metadata features strictly.
\end{enumerate}

\subsubsection{Scaling Regiments (RobustScaler)}
Standard Z-Score normalization fails aggressively navigating extreme non-normal traffic variances. Our architecture demands the Scikit-Learn \texttt{RobustScaler} implementation transforming variables $X$ uniformly anchoring boundaries over internal Interquartile Ranges (IQR).
\begin{equation}
    x_{scaled} = \frac{x - P_{50}(X)}{P_{75}(X) - P_{25}(X)}
\end{equation}
The computed constraints are specifically serialized to localized `.joblib` entities binding test-time execution identically.

\subsubsection{Supervised Vector Classification}

\subsubsection{Autoencoder Reconstruction Module}
Bridging deep parameters, the PyTorch implementation isolates normal vectors and maps complex nonlinear properties mathematically compressing limits exclusively scaling properties across the structural pipeline:
\[
    {d_{\text{features}}} \rightarrow 64 \rightarrow 32 \rightarrow 16 \rightarrow 32 \rightarrow 64 \rightarrow {d_{\text{features}}}
\]
Leveraging dropout limits minimizing overfitting gradients, the network calculates reconstruction matrices purely assessing $\mathcal{L}_{MSE}$ tolerances over internal dimensions.
\begin{equation}
    \min_\theta \frac{1}{N} \sum_{i=1}^{N} \|x_i - \hat{x}_i\|_2^2
\end{equation}
Extreme outlier calculations indicate the $95^{th}$ validation percentile forming the rigid boundaries denoting critical baseline breaks representing immediate zero-day triggers.

\subsubsection{Temporal LSTM Sequence Engine}
Single flows miss dynamic scanning events. The continuous PyTorch integration builds localized matrix limits representing preceding metadata $x$:
\begin{equation}
    \mathbf{X}_{seq} = [x_{t-9}, x_{t-8}, \dots, x_t]
\end{equation}
The engine tracks independent flows navigating heavy LSTM(64) boundaries generating Sigmoid regressions indicating immediate temporal density anomalies scoring natively $P_{\text{lstm}} \in [0,1]$. 

\subsubsection{PSI Concept Drift Mechanics}
Network topology continuously mutates. Validated environments measure population boundaries continuously tracking Expected bins ($E_i$) isolated during training phases against streaming Operational ranges ($A_i$).
\begin{equation}
    \text{PSI} = \sum_{i} \left( A_i - E_i \right) \times \ln\left(\frac{A_i}{E_i}\right)
\end{equation}
As $PSI_{avg}$ crosses boundaries signifying drift conditions, inference controllers actively scale evaluation probabilities dynamically throttling false warnings proactively adjusting threshold parameters directly via $\tau = f(\text{PSI})$.
